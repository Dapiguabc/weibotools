## 编译命令

* 输入命令 `$ npm start` 运行开发环境。
* 输入命令 `$ npm run runel` 运行开发环境的electron。
* 输入命令 `$ npm build` 编译到文件夹。
* 输入命令 `$ npm run asar` 编译asar文件。
* 输入命令 `$ npm run devdll` 编译开发环境dll文件。
* 输入命令 `$ npm run npmi` 或 `$ yarn run yarni` 安装生产环境依赖。
